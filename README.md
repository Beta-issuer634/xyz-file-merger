## ❓ Troubleshooting

### The file will not open

- Make sure the download finished
- Check that you opened the correct file
- Try running it again from your Downloads folder
- Right-click the file and choose **Run as administrator** if needed

### Windows blocks the app

- Select **More info** if Windows shows that option
- Choose **Run anyway** only if you trust the source
- Download the file again if it looks damaged

### The merge does not finish

- Check that you have enough free disk space
- Make sure the source files are not open in another app
- Try a smaller set of files
- Move the files to a local folder such as Documents or Desktop

### The output file is missing

- Look in the folder you chose before starting
- Search your computer for the file name
- Run the app again and pick a clear output folder
